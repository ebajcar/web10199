What you should already know

Web Development
  1. Create a properly structured web page (HTML5)
  2. Represent the Document Object Model (DOM)
  3. Recognize browser compatibility with standards.
  4. Style a web page (CSS3)
  5. Create page layouts using media queries.
  6. Publish a web page on the internet.

Key technologies

  1. HTML and CSS (user interface)
  2. Front-end languages (Javascript, EcmaScript, jquery)
  3. Back-end languages (node.js, python, java, PHP)
  4. Databases and web storage (SQL, MySQL, sessions)
  5. HTTP and REST (request, response, communication)
