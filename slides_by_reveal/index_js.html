JavaScript (JS)

  • a lightweight, interpreted (just-in-time compiled), programming language with first-class functions
  • a prototype-based, multi-paradigm, single-threaded, dynamic language
  • supports object-oriented, imperative, and declarative styles

JavaScript (JS)

  1. Standards are
    • ECMAScript Language Specification (ECMA-262) and
    • ECMAScript Internationalization API specification (ECMA-402)
  2. Documentation throughout MDN is based on the latest draft versions of ECMA-262 and ECMA-402.
