Learning Outcomes (from Course Outline)

  1. Design accessible forms that use major controls and form handling logic.
  2. Apply dynamically-typed variables with implicit type conversion to the creation of simple web apps using JavaScript.
  3. Choose JavaScript and Node.js functions and control structures (including selection and repetition) to the creation of simple web apps.
  4. Design small, accessible client-side web apps that employ JavaScript functions to respond to HTML DOM events, receive input from HTML form elements and produce output by making changes to the HTML DOM.
  5. Apply JavaScript's support for first class functions to the creation of simple client-side web apps.
  6. Prepare Node.js scripts that respond to HTTP requests containing GET and POST parameters to produce custom HTML content drawn from a one-table, multi-column MySQL database.
  7. Create Node.js scripts that respond to user requests by inserting, updating and/or deleting rows stored in a one-table, multi-column MySQL database.
  8. Produce Node.js scripts that make effective use of session management techniques.
  9. Apply best practices in internal documentation (including commenting, indenting and entity-naming), file naming and web site commenting, indenting and entity-naming), file naming and web site structure to the development of accessible web apps.
  10. Design accessible client-server web apps that make use of JavaScript on the client side for dynamic content and Node.js and MySQL on the server side for the generation of custom content.
