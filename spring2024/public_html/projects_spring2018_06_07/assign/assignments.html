Assignment Guide

Throughout the course, we build a members-only website offering games, surveys, and other activities for registered users.

Each assignment focusses on a specific set of web technologies, conforming to standards and coding conventions…

Phases of the project correspond to the assignments...note that assignments built on the previous, therefore they need to completed in the given order. Make sure you copy each phase so that you preserve the previous assignments and do not overwrite it with the new phase. The only copy that has the new changes is the current copy.
