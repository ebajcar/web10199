PHP Language Hightlights

  • Arrays: indexed and associative (compare to Javascript)
    $a = array();
    $a[0] = "first item";
    $b["carMake"] = "Honda";
    $c = array("first item", "second item", "third item");
    $d = array("firstKey" => "value associated with firstKey",
        "anotherKey" => "value associated with anotherKey");
  • Objects and classes (compare to Javascript)
  • Control structures: in addition to loops and decisions, make use of include and require
  • Error handling: explore die(), try-throw-catch, error_log, ...
  • PHP Functions & Methods

    empty($var)
    returns true if $var is empty array, string, or other data structure; otherwise false.

    strlen($str)
    returns the length of a string.

    isset($var)
    returns true if $var exists and has a value; false if it does not exist.

    is_numeric($var)
    returns true if $var contains a numeric value or can be case to a numeric value; otherwise false.

    trim($str), stripslashes($str), htmlspecialchars($str)
    functions to trim, remove slashes and replace HTML special characters with HTML entity codes; return a new version of the string passed as a parameter.

    rand($min,$max)
    returns a random integer from $max to $min; for example use rand(1,6) to simulate rolling a standard die.

    fopen(); fwrite(), fclose(), error_log(), file(), readfile()
    functions to read and write to text files stored on the server.

    session_start(); session_destroy();
    starts or resumes a session; terminates the session.

  • Database connection: use PDO object as preferred choice.

    new PDO("mysql:host=localhost;dbname=$DATABASE", $USERID, $PASSWORD)
    creates a connection handle to $DATABASE database using the given $USERID and $PASSWORD credentials; returns a PDO object representing the connection or throws an exception.

    prepare($command)
    PDO method which precompiles a given SQL query string and retuns a PDO Statement object. The query string can contain question marks for values to be filled in at execution time.

    execute()
    PDO Statement method to execute the prepared statement.

    execute($params)
    PDO Statement method to excute the prepared statement, using $params array to fill in the question marks in the prepared statement.

    fetch()
    PDO Statement method to fetch the next row from the dataset; returns the row as an associative array or false if there are no more rows.

two timers, one uses setTimeout to delay the callback to changeImg() to flip between the two images every two seconds; the second timer uses setTimeout to update the clock (callback runclk() every second. Also note the use of other DOM properties and methods (document.formName.inputName and document.images).

<body>
	<form name="mForm"> 
		<img src="rock.png">
		<input name="mButton" type="button" value="Click to start the timer" >
	</form>
<script>
	var id, ticker = 0, min = 0, sec = 0;
	clearTimeout(id);
	document.mForm.mButton.value = ""; 
	runclk();
	function runclk() {
		min = Math.floor(ticker/60);
		sec = (ticker-(min*60))+'';
		if(sec.length == 1) {sec = "0"+sec};
		ticker++;
		document.mForm.mButton.value = min+":"+sec; 
		id = setTimeout('runclk()', 1000);
	}
	
	var curImg = document.images[0];
	setTimeout('changeImg()', 2000);

		( curImg.src.search("rock") < 0 )
