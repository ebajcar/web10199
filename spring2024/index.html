Class Plan

What you should already know from last semester

Programming Foundations - Python PROG12583
  • Apply systematic problem-solving and analytical techniques in developing computerized solutions for simple scientific and/or business applications.
  • Determine the fundamental structural elements, their relationships and the execution control flow techniques used in small to medium-sized real-world programs.
  • Design and analyze execution control flow using various techniques such as flowcharts and debugging.
  • Characterize syntax and structures related to control statements and their use in computer programs.
  • Use and manipulate built-in (primitive) and user-defined data types.
  • Use and manipulate objects/resources for program implementation and enhancement.
  • Implement, execute, and debug programs in an appropriate development environment using industry best-practices and methodologies.
  • Construct programs that use major syntactic features of the programming language and a basic subset of common standard libraries.
Web Development SYST10049
  • Create a properly structured and accessible web page using (HTML).
  • Represent the Document Object Model (DOM) in a hierarchical diagram.
  • Recognize the importance of browser compatibility with HTML and CSS standards.
  • Style an accessible web page using Cascading Style Sheets (CSS) techniques.
  • Create page layouts that adapt to suit the constraints of the display using media queries.
  • Use internet protocols to publish a web site.
  • Explain how web servers display web pages.
  • Describe how to secure a web site.
  • Use design techniques to layout a user-friendly website.

These learning outcomes are copied from the respective Course Outlines. The course outlines provide you with guidance for what you need to learn during the course. If you have not mastered the material during the course, it is your responsibility to catch up without impacting others' learning experience. There is a lot of content to cover in each course and that leaves no time to learn something you should already have mastered. Now is the time to learn and practice deliberately the material currently being covered. If you need help, there is tutoring available — check out https://central.sheridancollege.ca/tutoring.

Course Learning Outcomes

  1. Apply dynamically-typed variables with implicit type conversion to the creation of simple web apps using JavaScript and PHP. ✔
  2. Apply JavaScript and PHP functions and control structures (including selection and repetition) to the creation of simple web apps. ✔
  3. Design small client-side web apps that employ JavaScript functions to respond to HTML DOM events, receive input from HTML form elements and produce output by making changes to the HTML DOM. ✔
  4. Apply JavaScript's support for first class functions to the creation of simple client-side web apps. ✔
  5. Create PHP scripts that respond to HTTP requests containing GET and POST parameters to produce custom HTML content drawn from a one- table, multi-column MySQL database. ✔
  6. Create PHP scripts that respond to user requests by inserting, updating and/or deleting rows stored in a one-table, multi-column MySQL database. ✔
  7. Create PHP scripts that make effective use of session management techniques. ✔
  8. Apply best practices in internal documentation (including commenting, indenting and entity-naming), file naming and web site commenting, indenting and entity-naming), file naming and web site structure to the development of web apps. ✔
  9. Design client-server web apps that make use of JavaScript on the client side for dynamic content and PHP and MySQL on the server side for the generation of custom content. ✔
