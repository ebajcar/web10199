Event attributes
  • obtrusive, does not comply with PE principles
  • does not meet ARIA standards, not supported by assistive devices
  • hard to debug and maintain code, code harder to read, inflate the markup, events nee to fire in JS, not in HTML code
Event methods &mdash using onevent properties
  • event handler: an object (function is an object) that contains an event handler functionality
Event listeners
  • most flexible way to set event handler is to use EventTarget.addEventListener.
  • allows multiple listeners to be assigned to an element and to remove listeners using EventTarget.removeEventListener. (i.e., the same button can perform different actions in different circumstances.); cleaning up old and unused event handlers can improve efficiency.
