Unobtrusive Javascript

  1. delay processing — place at the end body
  2. accelerate processing — DOMContentLoaded event
  3. avoid control boxes (alert, prompt,...)
  4. avoid event attributes — hard to maintain
  5. ultimately, move code to an external file

Key elements of programming style

  1. Expression
  2. Control structure
  3. Programming structure
  4. Input and Output
  5. Efficiency and instrumentation
  6. Documentation

Expressing each statement well:

  1. write clearly.
  2. be sparing with temporary variables.
  3. be unambiguous - parentheses, well-named identifiers.
  4. make sure conditional tests read clearly.
  5. avoid unnecessary branches.
  6. replace repetitive expressions by calls to a common function.
