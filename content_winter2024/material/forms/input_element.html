Input element

Each input element must include a type attribute to specify its component type, such as

type="text"
type="password"
type="checkbox"
type="date"
type="time"
type="color"
type="button"
type="hidden"
type="image"
type="file"
type="url"
type="email"
type="number"
type="radio"
type="range"
type="search"
type="submit"
type="reset"
type="tel"

The default is type="text", omitting the type attribute.


<!-- text type input -->
<form>
  <p><input></p>
  <p><input type="text"></p>
</form>

Both controls appear to be the same; omitting the type defaults to text type. input element is an empty element and its default behaviour is inline. Paragraph container was used to place them on separate lines.

An input tag can include name and value attributes to specify data for submission as a NAME=VALUE pair.

name value

Other attributes:

autofocus
checked
disabled
max
maxlength
pattern
placeholder
readonly
required
size
spellcheck
