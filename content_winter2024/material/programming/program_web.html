Web programming

What is web programming?

Definition Web programming is computer programming, a process that starts with an original formulation of a problem and culminates in executable computer programs. Computer languages, like human languages, require you to use them in order to understand them.

The process consists of a set of activities:

  1. Understand the problem. (analysis)
  2. Plan the logic. (design)
  3. Code the program. (construct)
  4. Translate program into machine language.
  5. Test the program. (test)
  6. Put program into production.
