Web Programming Examples

Board

Click "change state" button to move to next state.

  1. Initial state is set to -1. Hover over the table cells to see the background change to pink, then change back to white as you stop hovering over the cell.
  2. Press the change state button to change the state to 0. The first cell displays lightblue background and purple text. The remaining cells display red content.
  3. Press the change state button to change the state to 1. The first cell still displays lightblue background and purple text. The remaining cells display blue content.
  4. Press the change state again to change the state to 2. The first cell still displays lightblue background and purple text. The remaining cells display black background and white content.
  5. You can continue to change state to cycle through the three states.
  6. In each state, when you hover over the cells, note that only the first cell will change the background to pink, but not the others. The first cell is styled using a class selector (.lightblue), which has a lower specificity (10) than the pseudo-class selector (td:hover has specificity of 11). The remaining cells use inline style (style attribute) which has a specificity of 100 and therefore will always win over classes and pseudo-classes.
  7. Press start over button at any time to start from the beginning (reloads the page).
