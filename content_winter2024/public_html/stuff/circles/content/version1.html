Math Club Area of a circle

Problem Statement
Write a simple program that computes the area of a circle.

Version 1— This version hardcodes input and sends output to the console.

Algorithm
  1. Obtain the value of the radius of a circle
  2. Calculate the area of the circle using the given radius value
  3. Display the value
Constraints and notes
  1. values are constrained to integers and reals only (source: https://en.wikipedia.org/wiki/Number)
  2. the formula for calculating is area = πr2 where π=3.14159 (source: https://www.wikihow.com/Calculate-the-Area-of-a-Circle)
  3. output is sent to the console
  4. Can a radius have a negative value? No, it is a DISTANCE from center to the circumfrance. Does it need to be addressed? What if the user provides a negative value? Do we care? In this calculation it does not matter, because when you square a negative number, the result will be positive.
  5. what if we get an invalid value? need to test for valid and invalid values.

Simple solution (code)

<script>
// declare and initialize variable radius; assign integer or float values only
let radius = 5.7;

// declare variable circleArea that will store the calculated value and 
// assign it the formula π*radius*radius
let circleArea = Math.PI * radius * radius;

// display the calculated value and the radius value in the console
console.log("The area is ", circleArea, " when radius is ", radius);
</script>
</body>
Notes

• you can also use the Math.pow(radius,2) method which is equivalent to radius * radius
