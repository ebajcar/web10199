REGEX (Regular Expressions)

Pattern Attribute

The core HTML input controls support a pattern attribute that allows you to apply a regular expression to validate user input. The input element's value is checked against the pattern on form submission. Use the global title attribute to describe the pattern to help the user.

The pattern attribute works with the following input types: text, date, search, url, tel, email, and password.
