BOM - explore window properties and methods

  • querySelector vs. getElement* DOM methods

    item oneitem two

  • We start with a single-row table with two cells:

    <table>
    <tr>
        <td>item one</td>
        <td>item two</td>
    </tr>
</table>

    Using DOM methods, we add another cell, containing "added item": 

    80  var x = document.querySelectorAll("td");  /* nodelist of td elements */
81  console.log("81 x " + x + " x length " + x.length);
82  var y = document.getElementsByTagName("td"); /* HTML collection of td elements */
83  console.log("83 y " + y + " y length " + y.length);
84  var parent = document.getElementsByTagName("tr")[0];
85  var newItem = parent.appendChild(document.createElement("td"));
86  newItem.innerText = "added item";
87  console.log("newItem "+ newItem);
88  console.log("88 y " + y + " y length " + y.length); /* HTML Collection updated */
89  console.log("89 x " + x + " x length " + x.length); /* NodeList NOT updated */

    Open the console to see the results. Note the difference in length of the number of TDs in the HTML collection and the NodeList. Why?
