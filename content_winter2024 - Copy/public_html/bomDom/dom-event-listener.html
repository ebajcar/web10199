Events - add a simple listener

  • Click anywhere on the table to toggle the value of the second TD - two/three.

    one
    two
    five
    nine

  • Code snippet

    // Function to change the content of t2
function modifyText() {
  const t2 = document.getElementsByTagName("td")[1];  //HTML collection
  if (t2.firstChild.nodeValue == "three") {
    t2.firstChild.nodeValue = "two";
  } else {
    t2.firstChild.nodeValue = "three";
  }
}
// add event listener to table
const el = document.getElementsByTagName("table")[0]; //HTML collection
el.addEventListener("click", modifyText, false);
