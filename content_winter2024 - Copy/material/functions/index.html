Event handling

The Event interface represents any event which takes place in the DOM;
some are user-generated, while others are generated by APIs. Event itself contains the properties and methods which are common to all events.

Events allow you to elicit feedback from the user. If you want to execute any Javascript on a page, it must happen through an event. Examples of events: load a page, click a link, submit a form. Any event can have a function attached to it..
