Debugging Javascript Code

  • Use the console to trace values of variables
  • Use the console.log() method to trace values of variables and to send messages to the console at critical places in the code
  • It takes practice to understand the code. Use comments to explain, in your own words, what the code does and why
  • Use tools such as jshint.com to clean up Javascript code.
  • Review http://javascript.crockford.com/code.html
  • Here is a list of related suggestions covered in class
  • Compare console.log and document.write.
