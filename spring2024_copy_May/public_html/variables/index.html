Notes

  • A variable is a named memory location for storing a value.
  • There are eight basic data types in JavaScript.
  • Variables begin their life with a declaration. The remainder of their life within the application depends on where the variables are declared.
  • Examples: Order of events when page is loading
  • PE principle - avoid using control boxes (note the use of control boxes in the above example).
  • JavaScript ignores multiple spaces. You can add white space to your script to make it more readable.
  • For best readability, programmers often like to avoid code lines longer than 80 characters. The best place to break long lines is after an operator
  • Using document.write() method after an HTML document is loaded, will delete all existing HTML and should only be used for testing. w3schools Example
