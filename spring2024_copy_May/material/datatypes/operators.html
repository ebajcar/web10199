Operators are special symbols that represent computations like addition and multiplication. The values the operator uses are called operands.

Assignment

The basic = assignment operator should not be confused with the === equality operator, which compares two operands.

var x = 7;  // assigns the value 7 to variable called x

if ( x == "7" ) { 
  // the value of x will be compared with the value 7 (regardless of data type
}

if ( x === "7" ) { 
  // the value of x will be compared with the value 7 AND data types are compared
}

Arithmetic and Conditionals

var sum = 25 + 37;   // addition
var diff = 25 - 37;  // subtraction
var multi = 25 * 37; // multiplication
var div = 25 / 37;   // division
var mod = 25 % 5;    // modulus
var incr = x++;      // increment
var decr = x--;      // decrement

When the ++ increment or –– decrement operator prefixes the operand its value is changed immediately, but when they appear after the operand the expression is evaluated first

Arithmetic Shorthand

x += y;
// same as
x = x + y;

Combined assignment operators +=, –=, *=, /=, and %=, each perform an arithmetic operation then assign its result.

// The += operator is useful to append to an existing string.
var str = "The quick brown fox jumps ";
str += "over the lazy dog.";

Comparison operators

Two operands can be compared for === equality, !== inequality, > greater-than value, or < less-than value

var y = "3"; 
var x = 3; 
y == x      // returns true
y === x     // returns false

The <= and >= combined comparison operators also return true when both operands are equal

Expressions being evaluated for a true or false value need not include === true as that assessment is made automatically

Boolean Logic operators

The logical && AND operator evaluates two operands and returns true when both operands are true, but the logical || OR operator returns true when either operand is true. The logical ! NOT operator can prefix a single operand to return its inverse value

Conditional (ternary) operator

Conditional operator ?: evaluates a condition for a true or false value then executes one of two specified statements according to the result

// condition ? value1 : value2;
var status = (age >= 18) ? 'adult' : 'minor';

Concatenation and addition

In Javascript, you can add strings and numbers together. If you are doing math, the values must be numbers and not strings. The only actions you can do with strings is combine (concatenate) and compare. You can add, subtract, multiply, divide, and combine (concatenate) numbers. 

// save my name to a variable using the = operator
var myName = "Ellen";
// add a variable to a string
console.log( "My name is " + myName );
// add numbers together
console.log( 100 + 50 );
// concatenate strings together
console.log( "100" + "50" );

Priority (precedence)

  • Operator precedence can be set by enclosing an expression within parentheses to override the default operator priority
  • Where a statement contains nested expressions within parentheses the innermost expression gets evaluated first
  • Operator priorities are mostly like Java and C languages. For example, multiply, divide, modulus have priority over add and subtract; boolean and has priority over boolean or.
  • Brackets force priority.
var x = 100 + 50 * 3; // results in x having value 250
var x = (100 + 50) * 3; // results in x having value 450
