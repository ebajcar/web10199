Complete What is Node.js? - Intro to Node.js
What is Node.js? a runtime environment, full-stack development, APIs for basic HTTP communication, basic encryption, and other useful components.
What is Express.js? popular framework for Node.js; To use Express.js, we write a Node.js application, import Express.js, and then use the Express.js modules and functions we are interested in.
How Node.js Works. run on the server, independent of what JavaScript engines are used by browsers. Uses a single event loop. (Event Queue, Thread Pool)
Why Learn Node.js? Performance, Unopinionated, full-stack development, NPM, Scalability, Portability, Dev community, Frameworks
