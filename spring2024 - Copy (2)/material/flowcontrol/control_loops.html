Control structures — Iteration (Loops)

  1. for - loops through a block of code a number of times 
    for ( initializer, condition, modifier ) { 
  statements-to-be-executed 
}
  2. for…in - loops through the properties of an object
  3. while - loops through a block of code while a specified condition is true 
    initializer; 
while( condition ) { 
  statements-to-be-executed ; 
  modifier ; 
}
  4. do…while - also loops through a block of code while a specified condition is true
  5. A break statement immediately exits a loop
  6. A continue statement stops execution of a single iteration
