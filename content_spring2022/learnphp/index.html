PHP Language (server-side programming)

What distinguishes PHP from something like client-side JavaScript is that the code is executed on the server, generating HTML which is then sent to the client. The client would receive the results of running that script, but would not know what the underlying code was.

Introduction to PHP

PHP Hypertext Preprocessor (PHP) is an open source server-side scripting language designed for web develoment.

  • a PHP script is executed on the server, and the plain HTML result is sent backtot he browser.
  • you need to have solid understanding of HTML, CSS, and Jaascript (PE approach applies)
  • a PHP script starts with <?php and ends with ?>
  • File extension for PHP files is filename.php
