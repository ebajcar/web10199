Javascript language

Unobtrusive Javascript

  1. delay processing — place at the end body
  2. accelerate processing — DOMContentLoaded event
  3. avoid control boxes (alert, prompt,...)
  4. avoid event attributes — hard to maintain
  5. ultimately, move code to an external file

Test yourself

  1. After you complete any program, put it aside.
  2. Recreate the solution from scratch, without looking at the solution.
  3. When you get stuck, reflect on why. Determine what stopped you from completing it. Study related material (and yes, you can look at the solution to help you figure it out at this point).
  4. Repeat steps 1, 2, and 3, until you can produce the solution without any help.

Unobtrusive JavaScript

  • Scripts placed after marked up content
  • Accelarated to DOM being ready
  • No event attributes
  • No control boxes

Functional paradigm

  • Functions call other functions
  • init() calls gameReset() and cellWasClicked()
  • No global variables
  • Values passed as arguments as needed
