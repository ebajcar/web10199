Area of a circle - develop a solution

  1. Problems
  2. area-problem.html

  • x

    Write a simple program that computes the area of a circle.

  • A console example solution

    //
let radius = 5.7;
let areaOfCircle = Math.PI * radius * radius;
console.log("The area is ", areaOfCircle, " when radius is ", radius);
    Notes• you can also use the Math.pow(radius,2) method

  • A procedural example solution

    //
function calculateAreaOfCircle(radius) {
    return Math.PI * radius * radius;
}
let radius = document.getElementById("radius").value;
let areaOfCircle = calculateAreaOfCircle(radius);
document.getElementById("answer").innerText = 
    "The area is " + areaOfCircle + " when radius is " + radius;
