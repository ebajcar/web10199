closures

  • Nested functions

    1. inner function full access to all the variables and functions defined inside the outer function
    2. the outer function does not have access to the variables and functions defined inside the inner function
    3. a closure is created when the inner function is somehow made available to any scope outside the outer function.

    Read more MDN web docs.

    function outside(x) {
  function inside(y) {
    return x + y;
  }
  return inside;
}
