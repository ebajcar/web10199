Node.js Project Structure
  • app.js is your main Node.js file, the starting point of your app
  • package.json contains information about the project (generated automatically when you run npm init)
  • package-lock.json used by NPM to keep track of your package installations and dependencies
  • /node_modules contains the files for any packages you installed
  • /public any directories or static files that are considered "public"
  • /routes contains all the .js files that perform routing tasks in your application
  • /views contains templates that are used to show HTML pages to a user with dynamic data
