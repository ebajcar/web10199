Activity 6.1. Review and preparation

Complete, review. dig deeper Worksheets 1, 2, 3, 4, and 5. Catchup from previous weeks.

Revisit the Class Plan Information and read about Active learning, Learning course content, Learning for mastery, not grades, How evaluation works, Formative & Summative Evaluation, etc.

It is important to continue working with Javascript. Practice regularly. Refactor code, explore different techniques to do the same thing, learn to evaluate your own work and identify how you can improve. Read and explore alternate problems and potential solution, practice communicating and collaborating with others. Write the same solution in both fully functional paradigm AND fully object-oriented paradigm.

Activity 6.2. STUDY Course material

Explore the following pages and the links provided on these pages to primary (PHP Manual) and secondary sources (w3schools). Compare to the Javascript language.

6.2.1. JS vs. PHP highlights
—Compare the important features of Javascript and PHP. What is same or similar? What is different?
6.2.2. Client-server dataflow
—Examine how the data flows in the client-server environment. Dow do you get data from HTML page to PHP environment? from PHP to HTML page? What role does HTML form play in the process?
6.2.3. PHP Language
—What is PHP? how does it fit with other web technologies and development strategies? What resources are available? primary source?
6.2.4. Grammar
—What are the start and end processing instructions? file extensions? pure and hybrid scripts? whitespace? case-sensitivity? commenting and documentation?
6.2.5. Data types
—What data types are supported in PHP?
6.2.6. Variables
—what are expressions? identifiers? constants? predefined variables? how do you declare variables? what variable scopes are supported in PHP?
6.2.7. Operators
—what is an operator? what PHP operator types are available? what is the operator precedence?
6.2.8. Program flow
—what are simple statements? selection control structures, iteration control structures? how you include and require external files in PHP?
6.2.9. Data structures
—what are indexed arrays? associative arrays? multidimensional arrays?
