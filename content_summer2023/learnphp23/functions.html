Arrays and Strings
Functions implode() and explode()

Function implode() —Convert an array to a string. Function explode() —Convert a string to an array.

An opportunity to compare to JAVA code
// JAVA
// create a string from an array of characters
// char[] charArray= {'H','e','l','l','o',' ','W','o','r','l','d'};
// String mesg3 = new String(charArray);
// System.out.println(mesg3);

// PHP: convert an array to a string
$charArray = array('H','e','l','l','o',' ','W','o','r','l','d');
echo implode("",$charArray);
echo "<br>";

// PHP: convert a string to an array
$str = "Hello world!";
$str2 = chunk_split($str,1,".");
print_r (explode(".",$str2));
Arrays and Strings
Function array_rand()
Function array_rand()
—Picks one or more random entries out of an array, and returns the key (or keys) of the random entries.
explore further at http://ca3.php.net/manual/en/function.array-rand.php
Code snippet from the rock-paper-scissors game:

// variable $moves is an indexed array, holding all possible 
// choices for the human user and the machine user (zero-based index)
$moves = array("rock", "paper", "scissors");

// array_rand($moves) selects one random entry from $moves array and
// returns the key of the select member. since this is an indexed array
// it will return the index.  The index is then used to retrieve the
// value at that location of the array
$machine = $moves[array_rand($moves)];

Function strcmp()
Function strcmp()
—Binary safe string comparison

The dictionary that the PHP interpreter uses for comparing strings are the ASCII codes for characters. This puts numerals before letters, and orders the numerals from 0 to 9. It also puts uppercase letters before lowercase letters.
