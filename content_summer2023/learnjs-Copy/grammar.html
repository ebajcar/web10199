Basic grammar
  • JavaScript is case-sensitive and uses the Unicode character set.
  • In JavaScript, instructions are called statements. Spaces, tabs and newline characters are called whitespace.
  • Each JavaScript statement must be terminated by a semi-colon.
  • Javascript code may also include helpful code comments.
  • The parser ignores everything between
    • /* and */ characters, and
    • // characters and the end of a line.

JavaScript follows the ECMA-262 standard..

Debugging Javascript Code

  • Use the console to trace values of variables
  • Use the console.log() method to trace values of variables and to send messages to the console at critical places in the code
  • It takes practice to understand the code. Use comments to explain, in your own words, what the code does and why
  • Use tools such as jshint.com to clean up Javascript code.
  • Review http://javascript.crockford.com/code.html
